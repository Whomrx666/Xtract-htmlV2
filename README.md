# Xtract-htmlV2 
![Xtract-htmlV2 preview](Xtract-htmlV2.jpg)

## introduction
Xtract-htmlV2 is a tool for getting the HTML code from the website you want and is the successor to the previous version

## Instalations
```
$ pkg update -y && pkg upgrade -y
$ apt update -y && apt upgrade -y
$ pkg install git
$ pkg install python -y
$ git clone https://github.com/Whomrx666/Xtract-htmlV2.git
$ cd Xtract-htmlV2 
$ pip install -r requirements.txt
$ python Xtract-htmlV2.py
```

## Instructions
- **first**: Install tools according to the instructions above
- **second**: Enter the complete target address with https or http
- **third**: Don't forget to put robot.txt behind the target domain address
- **last**: The tools will describe the html code of the target website
